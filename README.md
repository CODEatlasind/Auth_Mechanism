# Express Authentication Playground

A hands-on repository for learning and implementing different authentication mechanisms in **Express.js** using **JavaScript**.

The goal of this repository is to understand how various authentication strategies work under the hood by implementing each one from scratch wherever possible, comparing their use cases, advantages, disadvantages, and security considerations.

## Authentication Methods

- [x] Session-Based Authentication
- [ ] Basic Authentication
- [ ] JWT (JSON Web Token) Authentication
- [ ] OAuth 2.0
- [ ] OpenID Connect (OIDC)
- [ ] Passport.js Strategies
- [ ] API Key Authentication
- [ ] Cookie-Based Authentication
- [ ] Refresh Token Authentication
- [ ] Multi-Factor Authentication (MFA)
- [ ] Social Login (Google, GitHub, etc.)

## Learning Goals

- Understand how HTTP authentication works.
- Learn the difference between authentication and authorization.
- Understand cookies, sessions, and tokens.
- Learn how browsers store authentication data.
- Implement authentication without relying entirely on libraries.
- Explore real-world authentication flows used by modern applications.
